# 作品コンセプト

## 一文で表すと

「展示を見るために歩いた道が、すでに展示だった。」

## 体験

1. キャンパス入口でQRコードを読み取る
2. 利用目的を確認し、位置情報取得に同意する
3. キャンパス内を自由に歩く
4. 展示場では参加中の人々を抽象化した3DCGとして見る
5. 展示場入口で到着チェックインする
6. 自分の移動軌跡が10〜20秒に圧縮されて再生される
7. 軌跡が集合アートへ溶け込む

## 表現への変換

- 現在位置: 光、粒子、生命体
- 移動速度: 光の勢い、尾の長さ、音のテンポ
- 滞在時間: 光の大きさ、植物の成長
- 曲がった回数: 軌跡の揺れ、音色の変化
- 他者との接近: 共鳴、発光、波紋
- 過去の軌跡: 徐々に消える残光、地形の変化

## 展示モード

- 通常時: キャンパス全体を漂うカメラで参加者を表示
- 到着時: 対象者の入口から展示場までを追跡再生
- 集合表示: 一日の軌跡を重ね、キャンパスの活動を可視化
