# プライバシー設計

本作品は位置情報を扱うため、表現より先に参加者の安全と選択権を設計する。

## 原則

- 参加は完全に任意とする
- 位置取得前に目的、範囲、保存期間を説明する
- 個人名、学籍番号、メールアドレスを収集しない
- 匿名IDは作品参加以外のデータと結合しない
- 参加者がいつでも追跡を停止できる
- 展示場到着後は自動的に追跡を終了する
- キャンパス範囲外の位置情報を保存しない
- 展示では座標をぼかし、個人を直接追跡できない表現にする
- 保存期間終了後に位置点と軌跡を削除する

## 同意画面に含める内容

- 作品の目的
- 取得する情報: 位置、精度、時刻、匿名ID
- 取得しない情報
- 展示での利用方法
- 追跡停止方法
- 保存期間と削除方法
- 問い合わせ先
